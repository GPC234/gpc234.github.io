如何使用完整安装包升级您的论坛
推荐的升级方法就是使用完整安装包。如果你所安装的 phpBB 系统中的核心文件曾经被你修改过，你也许想要用进阶升级包来升级，以保留那些你所做的修改。你也可以采用在 INSTALL.html 里所列出的方法来升级。使用 phpBB 3 的完整安装包来升级的步骤是：
备份所有的论坛文件和数据库。
到 phpBB.com 下载页面 下载最新的 "Full Package" 完整安装压缩包文件。
解压缩下载的压缩包文件。
从解压后的文件中 （不是你的网站），删除 config.php 文件， 以及下面几个文件夹 /images， /store 和 /files 。
登录你的论坛的管理员控制面板 ACP，Board settings 论坛设定，确认使用的默认风格是 prosilver 。
在你的网站的论坛根目录下，删除 /vendor 和 /cache 文件夹。
通过 FTP 或者 SSH 上传之前处理好的解压后的文件和文件夹到你的论坛的根目录，覆盖原有的文件，（注意：不要删除在你的 /ext 文件夹下的任何扩展。）
把浏览器指向你的升级目录，开始升级过程。
按照步骤升级数据库并且等待完成。
通过 FTP 或者 SSH 删除 /install 文件夹。
现在你就拥有了一个全新升级的论坛，并保留了你原有的用户和帖子。 后续的任务：

如何使用进阶升级包升级您的论坛
进阶升级包只推荐给专业用户使用，且当 phpBB 的核心文件曾经被修改过的情况下，才。您也可以采用 INSTALL.html 文档中列出的方式升级。进阶升级 phpBB3 的步骤是：
到 phpBB.com 下载页下载 "Advanced Update Package"。
解压缩档案.
上传解压完的 "install" 和 "vendor" 文件夹到您的 phpBB 根目录(config.php文件所在目录)。
一旦上传普通用户将无法访问论坛，因为有 install 目录存在。
现在输入 install 地址开始更新程序。
按向导完成开始更新。完成后更新程序将给您发送一个通知。
