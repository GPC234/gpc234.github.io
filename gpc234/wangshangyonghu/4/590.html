苹果手机找回iD密码
使用其他人的 iOS 设备重设你的密码
确保设备使用 iOS 15 或更高版本。
借用其他人的 iPhone、iPad 或 iPod touch，并按照以下步骤操作：
打开“App Store”App。搜索“Apple 支持”并进行下载。
打开 Apple 支持 App。向下滚动至“支持工具”。
轻点“重设密码”，然后点击“其他 Apple ID”。
输入需要重设密码的 Apple ID。
点按“下一步”，然后按照屏幕上的说明操作，直到你收到 Apple ID 密码已更改的确认电子邮件。
在登录新设备时重设密码
确保设备使用 iOS 10 或 macOS Sierra 或更高版本。
在 iPhone、iPad 或 iPod touch 上
设置期间，当系统提示你使用 Apple ID 登录时，请轻点“忘记”链接。然后轻点“忘记了 Apple ID 或密码”，并按照屏幕上的步骤操作。
如果你在设置期间跳过了使用 Apple ID 登录，请打开“设置”，然后轻点“登录 [设备]” > “没有或忘记 Apple ID”，并按照屏幕上的步骤操作。如果你使用的是 iOS 10.2 或更早版本，请轻点 “iCloud” > “忘记 Apple ID 或密码”，并按照屏幕上的步骤操作。
在 Mac 上
设置期间，当系统提示你使用 Apple ID 登录时，请点按“忘记了 Apple ID 或密码吗”，并按照屏幕上的步骤操作。
如果你在设置期间跳过了使用 Apple ID 登录，请前往 Apple 菜单 >“系统偏好设置”或“系统设置”，然后点按 “iCloud”。点按“忘记了 Apple ID 或密码吗”，并按照屏幕上的步骤操作。

确保设备使用 iOS 15 或更高版本。

打开 Apple 支持 App。向下滚动至“支持工具”。
