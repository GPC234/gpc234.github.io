数学公式表

代数部分

一元二次方程的求根公式：对于方程 ax2+bx + c = 0（a≠0），其根为 x=(-b±√(b2-4ac))/(2a)

等差数列通项公式：an=a1+(n - 1)d，其中 a1是首项，d是公差

等差数列前 n项和公式：Sn=(n(a1+an))/2=na1+(n(n - 1)d)/2

等比数列通项公式：an=a1qn - 1，其中 q为公比

等比数列前 n项和公式（q≠1）：Sn=(a1(1 - qn))/(1 - q)

几何部分

圆的面积公式：s = πr2，其中 r是圆的半径

圆的周长公式：C = 2πr

圆柱的侧面积公式：S侧=2πrh，其中 h是圆柱的高

圆柱的体积公式：V = πr2h

圆锥的体积公式：V=(1/3)πr2h

三角函数部分

正弦定理：a/sinA = b/sinB = c/sinC

余弦定理：c2=a2+b2-2abcosC

两角和的正弦公式：sin(A + B)=sinAcosB + cosAsinB

两角差的正弦公式：sin(A - B)=sinAcosB - cosAsinB

两角和的余弦公式：cos(A + B)=cosAcosB - sinAsinB

两角差的余弦公式：cos(A - B)=cosAcosB + sinAsinB
