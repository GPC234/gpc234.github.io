如何将您的网站迁移到 InfinityFree
如果您的网站不使用 MySQL 数据库，那么移动您的网站非常容易。只需使用 FTP 或文件管理器将所有文件上传到主机帐户上的“htdocs”文件夹。如果您有一个基于 PHP 的网站，请检查它是否有任何需要设置域名或文件路径的配置文件。根据需要更新这些值。如果您的网站使用 MySQL 数据库，您还需要移动数据库内容并更新网站配置以使用数据库：
备份旧主机上的数据库。如果您当前的提供商有 MySQL 备份工具 - 使用它。通常，您可以登录 phpMyAdmin，选择您的数据库并单击“导出”选项卡。选择另存为文件并选择“gzipped”。之后应开始下载数据库备份。
在您的 InfinityFree 帐户上创建数据库。在恢复数据库备份之前，您需要在我们的系统上创建一个数据库。进入您网站的控制面板，然后单击 MySQL 数据库。您可以在那里创建一个新数据库。请密切注意数据库名称和主机名，因为稍后您将需要它们。
将数据库数据导入我们的服务器。创建数据库后，输入我们的 phpMyAdmin（该工具的图标位于控制面板中）。进入后，单击“导入”选项卡，浏览到数据库的备份文件（例如 yourdatabase.sql.gz），然后单击“开始”。数据库恢复程序将开始。请耐心等待，因为如果您的备份很大，可能需要很长时间才能完成。
更新配置文件。现在，当数据库恢复时，您需要打开 PHP 脚本的配置文件（例如，WordPress 将有一个 wp-config.php 文件）并更新数据库主机、数据库名称、数据库用户名和数据库密码值。可以在控制面板中找到数据库主机名、数据库名称和数据库用户名。数据库密码可以在客户区找到。注意：我们不使用“localhost”作为 MySQL 主机名，正确的值应该类似于“ sql123.epizy.com” 141”。
